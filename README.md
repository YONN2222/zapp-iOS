# Zapp für iOS

![ZAPP](fastlane/metadata/android/en-US/images/featureGraphic.jpg)

### Ich entwickle aktuell eine iOS Version von [Zapp](https://github.com/mediathekview/zapp). Ziel ist eine moderne App, die dir schnellen Zugang zu vielen öffentlich rechtlichen Sendern und Mediatheken bietet. Die App entsteht für iPhone und iPad und wird laufend erweitert.

## Über

Zapp für iOS bringt das Android App Erlebnis auf iOS Geräte. Sender wie ARD und ZDF werden unterstützt. Du kannst diese Sender live schauen, durch Mediatheken browsen und Inhalte direkt streamen.

Die App befindet sich im aktiven Aufbau und entwickelt sich stetig weiter. Feedback und Ideen sind jederzeit willkommen.

## Features

<ul>
    <li>iOS Client für Zapp</li>
    <li>Live TV von mehr als dreißig öffentlich rechtlichen Sendern</li>
    <li>Direkter Wechsel zwischen Sendern ohne Verzögerung</li>
    <li>Übersichtliche Programminfos für laufende Sendungen</li>
    <li>Komfortable Suche in den Mediatheken</li>
    <li>Stabile Wiedergabe im Hintergrund</li>
</ul>

**Hinweis:** Bestimmte Sender können im Ausland blockiert sein.

## Status

Die iOS Version befindet sich im aktiven Aufbau und entwickelt sich von Woche zu Woche weiter. Testversionen folgen, sobald die Grundfunktionen stabil laufen.
